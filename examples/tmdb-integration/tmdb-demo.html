FabricPop — TMDB Integration Demo

Movie & TV Show Lookup Demonstration

About This Demo

This is a standalone demonstration of TMDB (The Movie Database) API integration for FabricPop.

The demo uses the API credentials from your .env file and showcases:

  • Search functionality (movies, TV shows, or both)
  • Trending content
  • Popular movies
  • Rich media display with posters and metadata
