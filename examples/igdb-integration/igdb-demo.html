About This Demo

This is a standalone demonstration of IGDB (Internet Game Database) API integration for FabricPop.

The demo uses Twitch OAuth credentials from your .env file and showcases:

  • Game search functionality
  • Popular and top-rated games
  • Rich media display with covers and metadata
  • Automatic OAuth token management
